Temel Özellikler

Delta CLI ile neler yapabilirsiniz?

Sistem İzleme

Bilgisayarınızın CPU, RAM ve disk kullanımını gerçek zamanlı izleyin.

system -cpu

Ağ Tarama

Yerel ağınızı ve açık portları güvenli bir şekilde tarayın.

scan -network

Geliştirici Araçları

Git ve Docker entegrasyonu ile geliştirme iş akışınızı hızlandırın.

dev -git status

Proje Oluşturma

Yeni Python ve web projeleri hızlı bir şekilde oluşturun.

new -py myproject

Komut Kılavuzu

En sık kullanılan komutlar ve açıklamaları

system -ram
RAM kullanımını görüntüler
system -cpu
CPU kullanımını görüntüler
system -disk
Disk kullanımını görüntüler
system -processes
Çalışan işlemleri listeler
scan -network
Yerel ağdaki tüm cihazları tarar
scan -port @localhost
Localhost'taki açık portları tarar
ping 192.168.1.1
Belirtilen IP adresine ping gönderir
self -info -ip
Kendi IP adresinizi görüntüler
dev -git status
Git deposunun durumunu görüntüler
dev -docker ps
Çalışan Docker konteynerlerini listeler
dev -note add "Yapılacak işler"
Yeni bir not ekler
new -py myproject
Yeni bir Python projesi oluşturur
open .
Mevcut dizini VS Code'da açar
