Delta CLI Dokümantasyonu

Hızlı Başlangıç

Delta CLI, modern bir terminal deneyimi sunan güçlü bir komut satırı aracıdır. Python tabanlı bu araç, sistem yönetimi, ağ tarama ve geliştirici araçlarını tek bir arayüzde birleştirir.

Delta CLI'yi kullanmak için Python 3.8 veya üstü gereklidir.

Hızlı Kurulum

bash 
pip install delta-cli

Detaylı Kurulum

Delta CLI'yi farklı platformlarda kurma yöntemleri aşağıda açıklanmıştır:

Windows

Windows için birkaç farklı kurulum yöntemi mevcuttur:

1. Winget ile Kurulum

powershell 
winget install delta-cli

2. Python pip ile Kurulum

powershell 
pip install delta-cli

Sistem Komutları

Delta CLI'nin sistem izleme ve yönetimi için sunduğu komutlar:

CPU İzleme

bash 
system -cpu

CPU kullanımını çekirdek bazında görüntüler.

RAM İzleme

bash 
system -ram

RAM kullanımını detaylı olarak gösterir.

Ağ Komutları

Ağ tarama ve analiz işlemleri için kullanılan komutlar:

Ağ Tarama

bash 
scan -network

Yerel ağdaki tüm cihazları tarar ve listeler.

Bu komut yönetici izni gerektirebilir.

Geliştirici Komutları

Git, Docker ve diğer geliştirme araçları için komutlar:

Git Komutları

bash 
dev -git status

Git deposunun durumunu kontrol eder.

Docker Komutları

bash 
dev -docker ps

Çalışan Docker konteynerlerini listeler.
